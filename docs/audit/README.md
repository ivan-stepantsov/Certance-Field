# Audit pack

Reviewer-ready material for taking the Certance Token Kit through a security /
compliance review in a regulated environment.

- **[SECURITY-ONE-PAGER.md](SECURITY-ONE-PAGER.md)** - single-page summary for an InfoSec reviewer: data-flow table, security posture, and verification commands.
- **[DATA-FLOW.md](DATA-FLOW.md)** - data-flow diagram (what is local vs what contacts a model) with a per-path table.
- **[IP-COI-DISCLOSURE-TEMPLATE.md](IP-COI-DISCLOSURE-TEMPLATE.md)** - a template to disclose an externally-owned tool to your employer before evaluating or deploying it (not legal advice).
